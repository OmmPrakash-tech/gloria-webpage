

# 🛍️ Gloria - Fashion E-Commerce Frontend

A modern and responsive fashion e-commerce website UI built using HTML, CSS, and JavaScript. This project showcases a stylish online clothing store with smooth animations, product sliders, and an elegant user interface.

---

## 🚀 Features

- ✨ Modern and clean UI design
- 📱 Fully responsive layout (mobile, tablet, desktop)
- 🎞️ Hero section with animated slider
- 🛒 Featured products carousel
- 🧥 Product categories (Women, Men, Kids, Accessories, etc.)
- 🔍 Interactive navigation bar with icons
- ❤️ Wishlist and cart UI elements
- 📖 Blog/Journal section
- 📩 Email subscription form
- 🎯 Smooth animations and transitions

---

## 🛠️ Technologies Used

- HTML5 <br/>
- CSS3 (Minified styles)<br/>
- JavaScript (Vanilla JS)<br/>
- Swiper.js (for sliders)<br/>
